### 🔬 Research Interests

- **LiDAR-Inertial Odometry (LIO)** — tightly-coupled state estimation, loop closure & pose-graph optimization
- **SLAM** — real-time mapping with Cartographer, FAST-LIO and related front/back-ends
- **Autonomous Navigation** — path planning, complete-coverage maneuvering for ground & surface vehicles
- **Legged & Mobile Robots** — quadruped control and low-cost mobile platforms
